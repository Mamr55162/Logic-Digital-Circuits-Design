# Logic-Digital-Circuits-Design

## What is it?
A C++ implementation of both combinational and sequential logic circuits.

## What does it consist of?
This project focuses mainly on implementing ICs both combinational & sequential, starting from simple logic gates
moving to the more complex operations built on logic gates, then finally implementing a real-life IC that applies the wanted operation.

## What was used in this project?
I used C++ programming language with **JetBrains CLion** as the IDE, NI Multisim as a simulator.

## Features
- All logic gates are built using a universal gate **NAND**.
- ICs are implemented to mimic real-life circuits as accurately as possible.
- Each IC & circuit has a simulation designed in Multisim.
- Contains real life Indicators such as LEDs, probes, etc.

## How to run it?
until now, the only way to try it is through a terminal, I am currently still working on a GUI for the project.
