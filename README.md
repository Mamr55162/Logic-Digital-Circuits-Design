# Digital-Circuits-Design

## 🚀 Project Title
A C++ implementation & Multisim simulation of combinational and sequential logic circuits.

## 📖 Project Description
This project focuses mainly on implementing ICs both combinational & sequential, starting from simple logic gates
moving to the more complex operations built on logic gates, then finally implementing a real-life IC that applies the logic as needed.

## 🛠️ What was used in this project?
* [C++] - Core Language.
* [JetBrains CLion](https://www.jetbrains.com/ides/) - Integrated development environment (IDE) used.
* [NI Multisim](https://www.ni.com/en.html) - For Simulations & Designs.

## 📌 Features

- NAND-gate-based implementation of fundamental logic gates
- Custom implementations of:
  - NOT
  - AND
  - OR
  - NOR
  - XOR
  - XNOR
  - Tri-state buffer
- Digital combinational circuit simulation in C++
- Half Adder and Full Adder implementations
- 4-bit adder simulation (74HC283)
- 4-bit magnitude comparator (74HC85)
- Multiplexer implementations:
  - 4-to-1 MUX
  - 74HC151
- Decoder implementations:
  - 2-to-4 decoder
  - 74HC154
  - 74HC42
- BCD to 7-segment display decoder
- Binary signal visualization utilities
- IC-inspired architecture using real-world 74HC-series logic designs
- Modular and extensible circuit simulation framework
- Multisim circuit schematics included

## 💻 How to use it?
Until now, the only way to try it is through a terminal by compiling and running the .cpp file, I am currently still working on a GUI for the project.
